# VLF - LMS

## Quick Start 

### install dependencies
```
composer install
```

### create env file
```
cp .env.example .env
```

### generate env key
```
php artisan key:generate
```

### migrate the migration and seed the database
```
php artisan migrate:fresh --seed
```

### start server
```
php artisan serve
```
